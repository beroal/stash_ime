Stash IME, the application.

In this application, “character” means a Unicode code point, not a Java character. This input method editor is implemented by a service {@link ua.in.beroal.stash_ime.Ime}. The settings screen of this input method is implemented by {@link ua.in.beroal.stash_ime.EditKbSearchActivity}. Keyboard Editor is implemented by {@link ua.in.beroal.stash_ime.EditKbFragment}. Unicode Explorer is implemented by {@link ua.in.beroal.stash_ime.SearchCharFragment}. The internal clipboard is implemented by {@link ua.in.beroal.stash_ime.CharClipboardFragment}. The “CPV” screen is implemented by {@link ua.in.beroal.stash_ime.CpvActivity}.
